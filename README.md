# Atividades Práticas - Desenvolvimento com React

Repositório com as três atividades da disciplina:

1. **Formulário com validação** (React + Formik + Yup) – pasta `atividade-1-2-react`
2. **Testes unitários** (Contador com Jest/React Testing Library) – pasta `atividade-1-2-react`
3. **Calculadora de soma** (Next.js) – pasta `atividade-3-next-calculadora`

## Como rodar

### Atividades 1 e 2 (React)
```bash
cd atividade-1-2-react
npm install
npm run dev
```
Testes: `npm run test`

### Atividade 3 (Next.js)
```bash
cd atividade-3-next-calculadora
npm install
npm run dev
```
Acesse: http://localhost:3000
